How to connect

One-time setup

  1. Create a Google Sheet named Mansi Life Tracker.
  2. Open Extensions, then Apps Script.
  3. Paste the code from apps-script/Code.gs.
  4. Deploy as a Web App: execute as you, access anyone with link.
  5. Paste the Web App URL above and save.
  6. Push this folder to GitHub repo mansigoel98/life-tracker and enable GitHub Pages.
